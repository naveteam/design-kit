# Tag

O component de Tag permite o usuário executar ações ou mostrar informações destacadas. Também pode ser utilizado como marcações para filtros, selects e checkbox.

      </Flex>
    </ThemeProvider>
  </Story>
</Preview>

# Selected | Not selected
As variants selected e not selected são utilizadas para definir se um conteúdo está ou destacado.
