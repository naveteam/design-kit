# Checkbox

O componente `Checkbox` deve ser utilizado em casos no qual o usuário pode selecionar mais de uma opção dentre uma lista de opções. Em sua versão desktop os Checkboxes são mostrados lado a lado. No mobile, os Checkboxes são mostrados verticalmente. Em suas props padrões, o componente está habilitado e desmarcado.

# Checked | onChange

O `Checkbox` pode receber a prop `checked`. É um valor booleano que indica o valor inicial para um `Checkbox`, por padrão o valor é `false`. Se o valor `checked` for passado para o `Checkbox`, será necessário também passar uma função de onChange para mudar o seu valor, caso contrário, o valor de checked nunca irá mudar. Se o componente estiver desabilitado, não será necessário passar uma função de `onChange`.

# Disabled

A prop `disabled` desabilita completamente o `Checkbox`, impedindo seu uso e as cores customizadas não irão funcionar.
