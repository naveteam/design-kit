# Checkbox

O componente `Checkbox` deve ser utilizado em casos no qual o usuário pode selecionar mais de uma opção dentre uma lista de opções. Em sua versão desktop os Checkboxes são mostrados lado a lado. No mobile, os Checkboxes são mostrados verticalmente. Em suas props padrões, o componente está habilitado, desmarcado e exige que um texto seja fornecido para cada `Checkbox`, caso isso não ocorra, um texto padrão será exibido.

# Color

O `Checkbox` pode receber a prop `color`. Pode ser utilizada alguma cor definida no tema ou qualquer outra cor que esteja em formato aceito pelo CSS. Para usar uma cor customizada, o checkbox não pode estar em sua variant `disabled`. Caso a prop não for passada, o texto do checkbox ficará com a cor padrão.

# Checked | Disabled

Em conjunto com a prop `color`, o `Checkbox` pode receber também a prop `checked`. O valor desta prop indica o valor inicial para um `Checkbox`, por padrão o valor é `false`. A prop `disabled` desabilita completamente o `Checkbox`, impedindo seu uso. Cores customizadas não irão funcionar. Se algum valor de checked for passado para o componente, este dado será adicionado ao seu valor padrão.
