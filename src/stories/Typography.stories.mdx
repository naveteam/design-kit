# Typography

É o componente base para a criação de todos os outros tipos de tipografia. Sua função é prover um componente flexível e seguindo a definição do [styled-system](https://styled-system.com/api). Pode ser utilizado em situações específicas onde nenhum dos componentes de tipografia previamente desenvolvidos atendam os requisitos necessários.

# Headings

O componente de Heading é uma extensão do componente [Typography](#typography), onde as props `fontSize` e `as` são definidas baseadas no valor de `variant`. Todos os demais parâmetros passados para esse componente são transmitidos para o componente Typography, portanto todas as props do Typography também são válidas no componente de Heading.

# Subtitle

O componente de Subtitle é uma extensão do componente [Typography](#typography), com valores definidos para `as`, `fontSize` e `fontWeight`. Todos os demais parâmetros passados para esse componente são transmitidos para o componente Typography, portanto todas as props do Typography também são válidas no componente de Heading.
