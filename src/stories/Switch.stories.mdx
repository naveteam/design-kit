# Switch

O componente de `Switch` é utilizado para a seleção de dados na aplicação, assim o usuário pode interagir com seus estados de seleção e não seleção. Sua estrutura é feita com base em um input de `checkbox` recebendo então props de um checkbox padrão html.

<Preview>
  <Story

    parameters={{

      }
    }}
  >
    <ThemeProvider>
      <Flex>
        <Switch name='base' defaultChecked />
        <Switch name='disabled_on' disabled defaultChecked />
        <Switch name='disabled_off' disabled />

    </ThemeProvider>
  </Story>
</Preview>

Ele pode receber também, através da propriedade `label` um texto.
