# Icon

O componente de `Icon` é responsável por apresentar um dos ícones definidos em nosso Design System. Ele recebe um parâmetro principal chamado `icon` que define qual icon será renderizado.

  parameters={{
    design: {
      type: 'figma',

  }}
>
{IconBase.bind({})}
</Story>
