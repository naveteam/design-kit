# Tabs

O component de Tabs recebe uma lista de tabs como propriedade. A lista pode ser renderizada em um dos dois eixos, você define isso utilizando a propriedade `direction`
que aceita como valor `vertical` e, `horizontal`, que é o seu valor padrão.

Cada tab na lista deve ser um objeto que pode conter as seguintes propriedades: `label`, `icon` e `disabled (que desabilita a aba)`.
Por padrão o valor de `icon` é `false` e o valor de `label` é `Tab`.

O conteúdo de caba tab deve ser passado como `children` e deve estar na mesma ordem das `Tabs`.

Na versão mobile de `Tabs` verticais é renderizado uma `Tablist` como menu, estando alinhada à esquerda e
com o seu conteúdo escondido.

A visibilidade do conteúdo dessa versão é atualizada ao clicar no ícone de menu.
