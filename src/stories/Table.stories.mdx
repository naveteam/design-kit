# Tabela

O componente `Table` permite que sejam criadas tabelas estilizadas para apresentação de dados na aplicação. Para compor uma tabela utilizando o Saturn você deverá seguir os padrões de [sintaxe da nativos do html](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/table), para snippet's mais tangiveis, basta inspecionar um dos exemplos abaixo.

## Tabela `regular`
O tipo padrão de tabela é a `regular` que se diferencia do outro tipo disponível devido a borda externa que a compõe.
