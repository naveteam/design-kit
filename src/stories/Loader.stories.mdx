# Loader

O componente `Loader` é utilizado para apresentar carregamentos de forma visual ao usuário. Estando disponível com os tamanhos `major` (padrão) e `icon`. Juntamente com as variants com tempo definido (`determinate`) que possuí opcionalmente porcentagem e a variant sem tempo definido (`indeterminate` - padrão).

# Size

O `Loader` possuí a prop `size` que pode receber dois valores possíveis, `icon` e `major`, que indicam o tamanho do `Loader`, por padrão é `major`.

# Percentage

O `Loader` tem a prop percentage, que define a quantidade de carregamento já realizada, em conjunto com esta prop está `showPercentage`, um valor booleano que define se a porcentagem será mostrada ou não em forma de texto.
