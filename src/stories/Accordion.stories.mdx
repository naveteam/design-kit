# Accordion

O componente `Accordion` permite que uma área seja encolhida e/ou expandida. Deve ser utilizado para agrupar ou ocultar regiões complexas, mantendo a página limpa. A Sua utilização baseia-se na importação e utilização de 4 componentes. O `AccordionsWrapper` deve ser o container em volta de todos os Accordions, cada item consiste (Nesta Ordem) de um componente `Accordion` que você deve colocar como filhos dele o componente `AccordionHeader` que define o seu título, é também onde se encontra o local clicável para `abertura e fechamento do Accordion` e o conteúdo (children) do Accordion sendo definido no componente `AccordionDetail`.

O `AccordionHeader` extende os valores das funções [backgrounds](https://xstyled.dev/docs/style-props/#layout) e [layout](https://xstyled.dev/docs/style-props/#layout) do [xstyled](https://xstyled.dev/), o `AccordionDetail` extende as funções [backgrounds](https://xstyled.dev/docs/style-props/#layout) e [space](https://xstyled.dev/docs/style-props/#space).

# Expanded

É possível definir o estado inicial de aberto/fechado do accordion, para isto basta utilizar a prop `expanded` com o valor desejado, por padrão é `false`. O accordion controla o seu estado de aberto/fechado.
