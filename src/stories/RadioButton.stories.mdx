# RadioButton

O componente `Radio Button` deve ser utilizado em casos no qual o usuário pode selecionar apenas uma opção dentre uma lista de opções. Em sua versão desktop os Radio Buttons são mostrados lado a lado. No mobile, os Radio Buttons são mostrados verticalmente. Em suas props padrões, o componente está habilitado e desmarcado.

# Checked | onChange

O `Radio Button` pode receber a prop `checked`. É um valor booleano que indica o valor inicial para um `Radio Button`, por padrão o valor é `false`. Se o valor `checked` for passado para o `Radio Button`, será necessário também passar uma função de onChange para mudar o seu valor, caso contrário, o valor de checked nunca irá mudar. Se o componente estiver desabilitado, não será necessário passar uma função de `onChange`.

# Disabled

A prop `disabled` desabilita completamente o `Radio Button`, impedindo seu uso e as cores customizadas não irão funcionar.
