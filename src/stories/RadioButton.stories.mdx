# RadioButton

O componente `Radio Button` deve ser utilizado em casos onde o usuário pode selecionar apenas uma opção dentre uma lista de opções.
Em sua versão desktop os Checkboxes são mostrados lado a lado. No mobile, os Checkboxes são mostrados verticalmente.
É composto por subcomponentes e propriedades que podem ou não ser acrescentados à sua estrutura principal, de acordo com a necessidade e preferência do usuário.
Sendo eles:

- `name`: Define o nome do `input`. Obrigatória para sabermos quais botões pertecem a este `name`.
- `label`: Texto do `input`, recebe uma string que fica posicionada ao lado do ícone.
- `value`: Define qual o valor do componente, pode receber uma string, um numeral ou até um booleano.
- `disabled`: Define se o componente está ou não habilitado, recebe um booleano.
- `defaultChecked`: Define se o componente está inicialmente marcado ou não, recebe um booleano.
- `checked`: Define se o componente está marcado ou não, recebe um booleano.
- `color`: Define a cor do texto referente àquele `Radio Button`, recebe uma string informando a cor.
- `onChange`: Define o que será executado quando o componente for modificado.
- `id`: Define o `id` do `input`.

# Disabled | Default Checked

A prop `disabled` desabilita completamente o `Radio Button`, impedindo seu uso e as cores customizadas não irão funcionar.
Já a prop `defaultChecked` define o componente como checado inicialmente.

export const DisabledAndDefaultCheckedStoryControls = args => (
  <SaturnProvider>
    <RadioButton label='No Default Checked' name='disabledDefaultCheckedProp' value='noDefaultChecked' {...args} />
    <RadioButton
      label='Default Checked'
      defaultChecked
      name='disabledDefaultCheckedProp'
      value='defaultChecked'
      {...args}
    />
    <RadioButton
      label='Disabled No Default Checked'
      disabled
      name='disabledDefaultCheckedProp'
      value='DisabledNoDefaultChecked'
      {...args}
    />
  </SaturnProvider>
)

# OnChange

A prop `onChange` é responsável por chamar uma função quando o valor do `input` mudar.

  <SaturnProvider>

      onChange={e => window.alert(e.target.value)}
      label='Botão Um'
      name='onChangeProp'
      value='Valor Um'
      {...args}
    />
    <RadioButton
      onChange={e => window.alert(e.target.value)}
      label='Botão Dois'
      name='onChangeProp'

# Color | ColorIcon

O `Radio Button` pode receber as seguintes propriedades para cores: `color` e `colorIcon`.
A propriedade `colorIcon` indica qual do ícone apenas, enquanto a propriedade `color` indica a cor do ícone e do texto.
Se o componente estiver desabilitado, a cor do ícone e texto será `gray.500`.

export const ColorStoryControls = args => (
  <SaturnProvider>
    <RadioButton label='Botão Comum' name='colorProp' {...args} />
    <RadioButton colorIcon='green.400' label='Botão Comum Ícone Verde' name='colorProp' {...args} />
    <RadioButton color='red.400' label='Botão e Texto Vermelho' name='colorProp' />
    <RadioButton color='purple.400' label='Botão e Texto Roxo' name='colorProp' />
    <RadioButton disabled={true} label='Botão Disabled' name='colorProp' />
  </SaturnProvider>
)
